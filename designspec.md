# Developer Specification: Pressure Washing Business Landing Page

## Visual Identity
- **Theme:** Dark, high-contrast, energetic.
- **Color Palette:**
    - Background: Deep Navy (#0B1120) / Black (#020617)
    - Primary Accent: Electric Cyan (#22D3EE)
    - Secondary Accent: Vibrant Blue (#3B82F6)
    - Text: Pure White (#FFFFFF) / Light Gray (#94A3B8)
- **Typography:**
    - Headlines: Bold, heavy-weight sans-serif (e.g., Inter, 800-900 weight).
    - Body: Clean, readable sans-serif (e.g., Inter, 400-500 weight).

## Section Layout & Components
1. **Hero Section:**
    - Full-viewport height.
    - Title: "UNLEASH THE CLEAN." (or similar bold headline).
    - Subtitle: High-pressure cleaning for modern homes.
    - CTA: "Call to Book Now" button (Electric Cyan).
    - **Animation:** CSS shimmer or SVG filter effect on text/background to mimic water movement.
2. **Services Section:**
    - 3-card grid (Mobile-first stacked layout).
    - Cards: Sidewalks, Driveways, Patios.
    - Visuals: Iconography or high-impact photography with cyan overlays.
3. **About / Why Us:**
    - Focus on speed, quality, and "Trust Signals" (e.g., Fully Insured, Local Expert).
4. **Final Booking CTA:**
    - Large, high-impact section.
    - Phone Number: 334-472-8402 (Primary focus).
    - Bold button: "Click to Call".

## Animations & Technical Details
- **Water Shimmer:** Use a CSS linear-gradient animation or a subtle SVG displacement map for the water motif.
- **Hover States:** Cyan buttons should glow or expand slightly.
- **Stacking:** Mobile-first vertical stack, switching to grid on larger screens.
- **Tech Stack:** Plain HTML5, CSS3 (Flexbox/Grid), and minimal Vanilla JS for the scroll animations and phone link handling.
